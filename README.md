# Lo-Beads: Color by Number

Aplicación de colorear por números con temática de beads de pixel art.

- Más de 120 patrones únicos de pixel art
- Sistema de beads realista con efectos 3D
- Paleta de colores HAMA auténtica
- 4 poderes especiales
- Museo personal
- Sin anuncios, sin compras, totalmente gratis
